General

How do I install UAP Rhino Plugin?

1. Download .yak file here.
2. Double click or drag-and-drop into Rhino 8
3. Restart Rhino 8

Where is the documentation for all the commands that exist? New

The documentation includes detailed descriptions of all the commands, along with a gif demonstrating how each command works in practice.

List of every command in the UAP Plugin?

It is located here.

How do I know I have the most up to date version? Updated

Future updates will include it being able to install using the -PackageManager command in rhino and being public to download. For now, please use the UAP command to see what version you have.

What is this plugin used for?

The UAP plugin introduces essential features designed to streamline workflows, including tools for layouts, modeling, and mesh analysis, among others. This also helps the team unify what is needed and can be used as well.

What version of Rhino does this support?

Currently only supports Rhino 8. Rhino 8 now currently uses Python 3, while Rhino 7 uses Iron Python 2.7. This makes it uncompatible with Rhino 7 without a total rewrite.

Any plugins/scripts for Grasshopper or Houdini?

Currently focusing on Rhino, as its the assesible program for everyone, but will make some example scripts for those softwares and HDAs for HoudiniFX.

Can I access the source code for each command?

Of course! The github has all the source codes. Just keep in mind only Clark can change the commands for rolling out the next version.
